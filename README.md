# Taller-Snake
Implementacion juego Snake usando hilos.


## Requisitos
- JDK 17 o JDK 21
- Windows 11 (recomendado)

## Configuración
1. Clona o descarga este repositorio.
2. Abre el proyecto en tu entorno de desarrollo (Eclipse, IntelliJ, etc.).

## Ejecución
1. Se debe tener JDK 17 o JDK 21 instalado.
2. Ejecuta el proyecto desde tu entorno de desarrollo o mediante la consola.
   
   
## Especificaciones
- La tasa de crecimiento de la serpiente es de 10 puntos por cada esfera de comida con la que colisione.
- El archivo de propiedades del proyecto se encuentra en: `src\resources\propsnake.txt`.
- El archivo serializado para ver el historial está en: `src\resources\Historial.ser`.
